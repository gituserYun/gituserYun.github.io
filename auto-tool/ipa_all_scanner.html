IPA All Scanner

.ipa 파일 하나로 URL / API-KEY / Scheme(딥링크) 스캔을 한 번에 실행하고, 3개 시트짜리 엑셀 결과를 로컬 폴더에 저장합니다. 파일은 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.

① 결과 저장 폴더 연결

브라우저 보안 정책상 웹페이지가 임의로 로컬 폴더를 만들 수 없어서, 최초 1회만 폴더 선택창에서 "C: (로컬 디스크)"를 선택해주세요. 그 안에 gituseryun_scan 폴더가 없으면 자동으로 생성되고, 있으면 그대로 재사용됩니다. 이후에는 브라우저가 연결 정보를 기억하므로 다시 선택할 필요가 없습니다 (권한 재확인만 나올 수 있음).

연결 안 됨
⚠ Chrome 또는 Edge 브라우저에서만 동작합니다 (File System Access API 필요). Safari/Firefox는 지원하지 않습니다.

② IPA 파일 선택

분석할 .ipa 파일을 선택하세요.

선택된 파일 없음

③ 스캔 시작

결과 파일명: [ipa파일명]-[YYYY-MM-DD].xlsx (같은 이름으로 재실행 시 덮어씀)
