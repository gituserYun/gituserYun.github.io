Deep Link PoC TEST

아래 입력창에 테스트할 URL을 입력한 뒤, 스킴 버튼을 눌러 딥링크를 트리거

참고) adb 명령어

terminal 
# Activity 실행
adb shell am start -n <package>/<activity>

# 예시
adb shell am start -n com.example.app/.MainActivity

# Action 지정 실행
adb shell am start -a android.intent.action.VIEW -d "https://example.com"

# Extra 데이터 포함
adb shell am start -n com.example.app/.TargetActivity --es "key" "value"

# Broadcast 전송
adb shell am broadcast -a com.example.app.ACTION_NAME

# Service 시작
adb shell am startservice -n com.example.app/.TargetService

# Exported 컴포넌트 취약점 점검 시 (권한 우회 테스트)
adb shell am start -n com.example.app/.ExportedActivity -a android.intent.action.MAIN
